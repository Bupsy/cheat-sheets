## Pane Management
- Split vertical
  - `Ctrl b, %`
- Split horizontal
  - `Ctrl b, "`
- Synchronize panes
  - `Ctrl b, : setw synchronize-panes`
- Navigate panes
  - `Ctrl b, <arrow keys>`

## Window Management
- Create new window
  - `Ctrl b, c`
- Next window
  - `Ctrl b, n`
- Previous window
  - `Ctrl b, p`
- Kill current window
  - `Ctrl b, &`
